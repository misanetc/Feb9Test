# Code of Conduct

## Our Pledge

We as contributors, maintainers, and community members of this project pledge to make participation in this community a respectful, inclusive, and harassment-free experience for everyone, regardless of age, body size, disability, ethnicity, gender identity and expression, level of experience, nationality, personal appearance, race, religion, sexual orientation, or any other personal characteristic.

We are committed to fostering a welcoming environment that supports learning, collaboration, and constructive contribution, especially for newcomers to open source.

## Our Values

This project values:
- Respectful communication and collaboration
- Openness to diverse perspectives and experiences
- Empathy toward others, especially learners and newcomers
- Accountability for our actions and words
- A shared responsibility to maintain a healthy community

## Expected Behavior

All community members are expected to:
- Be respectful, patient, and considerate in discussions and feedback
- Use welcoming and inclusive language
- Accept constructive criticism gracefully
- Focus on what is best for the community and the project
- Show empathy when interacting with others
- Ask questions and help others learn when possible

## Unacceptable Behavior

Unacceptable behavior includes, but is not limited to:
- Harassment, discrimination, or exclusionary behavior
- Offensive, insulting, or derogatory comments or personal attacks
- Trolling, intimidation, or sustained disruptive behavior
- Publishing others’ private information without explicit permission
- Sexualized language or imagery in any context
- Any conduct that would reasonably be considered inappropriate in a professional or educational setting

## Reporting Violations

If you experience or witness behavior that violates this Code of Conduct, please report it by contacting the project maintainer directly.

Reports can be made via:
- Email: msnetchie11@gmail.com
- A private message through the repository’s issue system (if available)

All reports will be reviewed promptly and confidentially. The privacy and safety of reporters will be respected.

## Enforcement

Project maintainers are responsible for enforcing this Code of Conduct. They will investigate reported incidents and determine appropriate responses based on the severity and context of the behavior.

Possible enforcement actions include:
- A private warning or request for behavior change
- Temporary or permanent removal from discussions or contribution privileges
- Blocking or banning a contributor from the project

Maintainers are expected to act in good faith, fairly, and consistently when enforcing this policy.

## Scope

This Code of Conduct applies to all project spaces, including but not limited to:
- GitHub repositories, issues, pull requests, and discussions
- Community communication channels
- Project-related events or collaborations

## Attribution

This Code of Conduct is informed by the Contributor Covenant and the community-building guidelines from opensource.guide, adapted to fit the goals and values of this project.
